# Tic Tac Toe

A simple and interactive Tic Tac Toe web application built using HTML, CSS, and JavaScript. This project demonstrates core front-end development skills including DOM manipulation, game logic, and responsive UI design.

## Live Demo

You can play the game here:
[Live Demo Link](https://yoshitayp.github.io/TicTacToeY/)

## Features

* Two-player gameplay (Player X vs Player O)
* Interactive game board with smooth hover effects
* Win detection with highlighted winning cells
* Draw detection
* Restart game functionality
* Clean and modern user interface
* Responsive and centered layout

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

## How to Run Locally

1. Clone the repository:
   git clone https://github.com/yoshitayp/tic-tac-toe.git

2. Open the project folder

3. Open `index.html` in your browser

## Project Structure

* index.html – Main file containing structure, styling, and logic
* README.md – Project documentation

## How It Works

* The board is generated dynamically using JavaScript
* Each cell listens for click events
* Players alternate turns between X and O
* The game checks for winning combinations after each move
* Winning cells are highlighted when a player wins
* If all cells are filled without a winner, the game ends in a draw
